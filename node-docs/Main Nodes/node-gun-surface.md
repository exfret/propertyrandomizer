---
description: Access to a gun on a given surface. This can be obtained via an item, or a vehicle that has said gun.
type: OR
---
There is one of these per gun prototype and surface combination. Because this can be attained with vehicles, whose placement can be surface-specific, this too must be surface-specific.

Prereqs:
- The [[node-operate-entity-surface]] for a vehicle with this gun on this surface.
- The [[node-item]] corresponding to this gun.
